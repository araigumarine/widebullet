# Widebullet

Widebullet is an API gateway with JSON-RPC.

# Status

Production ready.

## Requirements

Gaurun requires Go1.7.3 or later.

# Installation

```
go get -u github.com/mercari/widebullet/...
```

# Run

```
wbt -c config/example.toml
```
